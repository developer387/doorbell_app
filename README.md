## ğŸ“± Running the App

## 1. Mobile App

To run on Android or iOS Simulator:

**Android:**
```bash
npm run android
# OR
npx expo start --android
```

**iOS:**
```bash
npm run ios
# OR
npx expo start --ios
```

**Interactive Menu:**
You can also run the standard start command and press `a` for Android or `i` for iOS:
```bash
npx expo start
```

## 2. Web App

To run the web version in your browser:

```bash
npm run web
# OR
npx expo start --web --tunnel

```
